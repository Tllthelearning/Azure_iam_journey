What is Entra
Entra is the new name for Azure AD.
It is simply a placeholder for Identities we create in Azure cloud.
Technically It plays a central role in cloud management.
From Identies and token perspective, This is acting as IDP(identity Provider).
IDP is the component in token based authentication and authorization which provides ID token,Access token and Refresh token.
For a new comer or naive ,These may sound pretty absurd but with time all concepts will get clear.
For now, just try to hold on all these terminologies.
1 thing which you need to keep in mind that Tenant,subscription will get automatically created upon accessing portal.azure.com for the first time but to do real practicals around conditional access and getting P1/P2 trial license,we need to act smart.
If you simply go on reading documents online,It will make you go wild and big time confused.

Keep in mind very clear difference between external user and internal user.
If you see your upn with EXT, it means you are being treated as external user and even if you provide yourself as Glbal administrator, you will not be able to use trial license P1/P2 which is mandatory for conditional access usage.
